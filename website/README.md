# FxTS website

## Prerequisite

- api-extractor
  ```
  npm install -g @microsoft/api-extractor@7
  ```
- api-documenter
  ```
  npm install -g @microsoft/api-documenter@7
  ```

## Install or Update plugin markdown-plugin

```
$ make install
```

## Document build

```
$ make build
```

## Start

```
$ npm run start
```
